#Spring JPA

* Abrtact complex needed to access a database
* Abstraction of JPA and Hibernate, specification for accessing & persisting & managing data between java Objects.
* Hibernate is an implementation of JPA ( Java Persistence Api ) the most popular one, takes any java objects and 
  Map between Objects and DataBase with ORM ( Object relational Mapping )
* Spring Data JPA : make thinks super easy to make, reduce code to write, access data bases without any sql command.
Make integration between Spring and JPA easier.

##Annotations

@Entity : 
>Map the class to a table in database
> Best practice to add a name @Entity(name ="ClassName) 

@Table
>Specify the name of the table on DB
> > name ="TableName"
> 
> >uniqueConstraint : Unique constraints that are to be placed on the table,
> constraints specified by the Column and JoinColumn annotations and constraints
> >> @UniqueConstraint ( name ="", columnNames =""")

@Id :
>Identify a PrimaryKey

@SequenceGenerator:
> Specify a sequence generator, params :
> (
>>   name ="", name of the generator
> 
>>   sequenceName="", sequence name
> 
>>   allocationSize="1" of incrementation of sequence n+1
>
> )

@GeneratedValue :
>generated value for the id ( 
> 
>>Strategie : Sequence, 
> 
>>generator : the name of sequenceGenerator
> 
> )

@Column :
> Specify a column of the table
> 
> @Column (
> 
> >name : name of column
> 
> >nullable = False / True (Default)
> 
> >unique = True / False (Default) [Don't use it if using @UniqueConstraint]
> 
> >updated = False / True (Default)
> 
> >columnDefinition = "TEXT" / ...
> 
>)
